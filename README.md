# Lüneryard – Backyard-Ultra Live-Dashboard

FSST-Projekt. Eine Desktop-App (Electron), die mehrere **Strava-Beacon-Livekarten**
gleichzeitig zeigt und für ein **Backyard-Ultra-Rennen** automatisch Runden, Zeiten,
Pace und Ausscheidungen berechnet – mit einem Info-Streifen für alle Zuschauer.

## Ordnerstruktur

```
docs/   Dokumentation (Dokumentation.md, Dokumentation.pdf) + Präsentation
src/    Quellcode (Electron-App)
bin/    Kompiliertes Programm (lokal gebaut – siehe bin/README.md)
```

## Schnellstart

```bash
cd src
npm install
npm start
```

(oder `src/start.bat` doppelklicken)
